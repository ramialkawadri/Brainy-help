Sidebar & Navigation

The sidebar is your command center for navigating Brainy. It provides quick access to all major features and file management capabilities.

Sidebar Components

Navigation & Quick Actions

  • Home — View your study overview and heatmap
  • Search — Find cells across all your files
  • Settings — Customize app preferences
  • Sync — Manually synchronize your data across devices
  • Help — Access documentation

File Management Area

The central section displays your complete file and folder structure, allowing you to organize and manage your study materials.

FSRS Configuration

Customize spaced repetition settings for individual files or folders to optimize your learning experience based on your preferences.

Account Management

Sign in, sign up, or manage your account settings from the sidebar.

Collapsing the Sidebar

Click the arrow next to the Brainy title to collapse and expand the sidebar and maximize your working space.

File Management

Brainy provides a comprehensive set of tools for organizing your study materials:

Creating Files and Folders

  • New Folder — Right-click on any folder and select New Folder to create a subfolder for organizing related content
  • New File — Right-click on a folder and select New File to create a new study file

Organizing Your Files

  • Rename — Right-click a file or folder and select Rename to give it a more descriptive name
  • Move — Drag and drop files or folders to reorganize them into different locations
  • Delete — Right-click and select Delete to remove items (⚠️ Note: Deletions are permanent and cannot be undone)

Sharing and Backup

  • Export — Right-click a file or folder and select Export to save it to your disk for sharing with others
  • Import — Import files you've exported or received from others

Customizing FSRS (Spaced Repetition)

Optimize your learning by customizing the FSRS (Free Spaced Repetition Scheduler) algorithm:

  • Right-click on any file or folder and select FSRS Profile
  • Adjust algorithm parameters to match your learning style and retention goals
  • Settings can be applied to individual files or entire folders
  • Changes take effect immediately for future reviews

Keyboard Shortcuts

Speed up your workflow with these essential keyboard shortcuts:

Shortcut Action
Ctrl + \ Collapse/Expand the sidebar
Ctrl + N Create new file
DEL Delete the selected file
F2 Rename selected file
Ctrl + P Open settings
Ctrl + H Navigate to home
Ctrl + Shift + H Navigate to search
