Search & Find

Quickly locate any content across your entire study library with Brainy's powerful search functionality.

How to Search

  1. Click Search in the sidebar or press Ctrl + Shift + F
  2. Type your search query in the text field
  3. Press Enter to view all matching results

Search Results

Search results display all matching cells across your files. For each result: - Click to preview the cell content - Use the cell reference to jump directly to that card in the editor

Tip: From any search result, you can instantly navigate to the original file and cell using the focus tools!
